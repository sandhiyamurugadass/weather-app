# 🌦️ Live Weather App

A command-line weather app built with Python that fetches **real-time weather data** for any city in the world using the OpenWeatherMap API.

## Features
- 🌡️ Current temperature & feels like
- 💧 Humidity percentage
- 💨 Wind speed
- 🌤️ Weather condition description
- 🔁 Check multiple cities in one session

## Setup

### 1. Get a FREE API Key
- Go to [openweathermap.org](https://openweathermap.org)
- Sign up for a free account
- Go to **API Keys** tab and copy your key

### 2. Add your API Key
Open `weather.py` and replace:
```python
api_key = "YOUR_API_KEY"
```
with your actual key.

### 3. Run the app
```bash
python weather.py
```

## Requirements
- Python 3.x (no extra libraries needed!)
